# 📖 Another tool to lookup dict.cc!

Usage:
```
λ dictcc awesome
+--------------------+---------------------+
| English            | Deutsch             |
+--------------------+---------------------+
| awesome            | furchteinflößend    |
| awesome            | geil                |
| awesome            | fantastisch         |
| ...                |                     |
+--------------------+---------------------+
```

## 🏗️ Building

A `c++17`-compiler as well as `libcurl` is needed.

Compile it just by
```
make
```

## 🚫 Missing Features

Are any features missing? If so, just [ask for it in an Issue!](https://github.com/lquenti/dict.cc/issues).
